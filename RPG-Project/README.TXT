CMPT 370 - Group 4 - T12 - Character Generator RPG

Installation instructions:

If running from source code in IDE
  1. Download or Clone the project from 'https://git.cs.usask.ca/dms224/cmpt370/-/tree/master'
  2. Open the project using 'IntelliJ IDEA' (The project must be built using this IDE)
  3. Click the dropdown box beside the green arrow at the top of the screen
  4. Click 'edit configurations'
  5. Select 'Java SDK 17' as your JDK
  6. Choose 'sample.Main' as the main class


If installing from source code and creating JAR.exe
  1. Download or Clone the project from 'https://git.cs.usask.ca/dms224/cmpt370/-/tree/master'
  2. Open the project using 'IntelliJ IDEA' (The project must be built using this IDE)
  3. Create an 'Artifact' from the source code
      - Click 'File' on the toolbar
      - Click 'Project Structure' in that menu
      - Click 'Artifacts' in the side menu
      - Click '+'
      - Click 'JAR', then 'from modules with dependencies'
      - Select the Project, then select the main class 'sample.Main', press 'OK'
      - (Optional) Select a new output directory, otherwise the JAR will be created in the 'out' folder
  4. Build the JAR from the Artifact
      - Click 'Build' on the toolbar
      - Select 'Build Artifacts'
      - Select the project JAR, then select 'Build'
  5. The JAR file can be found in the 'out' folder of the project
      - Execute the project from there

If installing from the wiki:
  1. Navigate to 'Deliverable 9' of the wiki (https://git.cs.usask.ca/dms224/cmpt370/-/wikis/Deliverable_9)
  2. Navigate to 'Project JAR'
  3. Click and download the project
  4. Open the project


Enjoy, and thank you!!
